01 / 13

Step-by-step guide

Prefer to read? Here's everything the video covers. The workflow is: Upload → Calibrate → (mask & extract, or place points) → Refine → Export.

1. Upload your graph
  • Drag an image onto the canvas, or click the drop zone to browse.
  • PNG, JPG, BMP, WEBP, GIF and SVG are supported.
  • Mouse wheel zooms (centred on the cursor); Space+drag or the Pan tool moves around; Fit reframes the image.
2. Calibrate the axes required
  • Pick the Calibrate tool. Click two known points on the X axis (X1, X2) and two on the Y axis (Y1, Y2) — gridline intersections are ideal.
  • Type each point's real value into the boxes on the left.
  • Tick Log X / Log Y for logarithmic axes (values must be positive).
  • Press Apply Calibration. Add/Auto unlock and the bottom readout shows live data coordinates. Rotated/skewed scans are handled automatically.
3. Set up datasets
  • Each curve should be its own dataset. Use New dataset, click a name to rename it, click the swatch to change colour.
  • The highlighted dataset is active — new points (manual or auto) go into it.
4. Manual extraction — Add / Adjust / Delete
  • Add (A): click to drop points; a magnifier appears for precision.
  • Adjust (S): drag a point to reposition; select one and use arrow keys to nudge (Shift = bigger steps).
  • Delete (D): click a point to remove it. Ctrl+Z undoes.
5. Automatic extraction — mask the curve
  • Fence in just one curve: Pen paints (brush slider), Box drags a rectangle, Erase trims, View toggles the overlay, Clear mask resets.
  • Masking excludes the other series, gridlines, labels and legend — the biggest factor for clean results.
6. Automatic extraction — colour & distance
  • Click Foreground color, then click the curve to sample its colour.
  • Distance is colour-match tolerance — lower it (≈20–30) so faint edge pixels aren't matched.
  • Filter colors previews which pixels will be captured; the eye-slash button hides it.
7. Automatic extraction — algorithm, ΔX/ΔY & scan
  • Averaging Window traces lines/step curves; X Step gives one point per column.
  • ΔX / ΔY set point spacing in pixels (smaller = denser).
  • Scan: X → columns (horizontal runs), Y ↓ rows (vertical risers), Both for a full step curve.
  • Press Run — Extract Curve; repeat per dataset for multi-series plots.
8. Refine for accuracy
  • Auto gets ~95%. Keep the mask snug, lower Distance, use smaller ΔX/ΔY.
  • Finish with Adjust (nudge), Add (exact corners), Delete (strays).
  • For Kaplan-Meier curves the key points are the step corners — a few by hand often beats automation.
9. Validate in Live Preview & download
  • View your data as Scatter, Line, or Step (KM) in the right panel.
  • Click PNG to download a clean, labelled chart with a legend.
10. Export & save the project
  • Export → CSV, TSV, JSON, or copy to clipboard.
  • Project → Save stores image, calibration, datasets & settings as .json; reload with Project → Load.
